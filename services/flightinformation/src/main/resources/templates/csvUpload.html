This is a simple page for uploading csv file to the flightinfo service.

Example data is available at:

  • Local directory src/main/java/resources/data/
  • https://openflights.org/data.html

Upload Airports Data

The csv is posted as multipart data form to /airports/import/csv. An empty page will be displayed in case of a successful import.
Please select the csv file.

Upload Airlines Data

The csv is posted as multipart data form to /airlines/import/csv. An empty page will be displayed in case of a successful import.
Please select the csv file.

Upload flight flightConnections Data

The csv is posted as multipart data form to /flightConnections/import/csv. An empty page will be displayed in case of a successful import.
Please select the csv file.
